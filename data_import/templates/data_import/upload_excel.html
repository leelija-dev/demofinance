Required Excel Structure
Customer Information (Required)
  • customer_type (NEW/EXISTING)
  • full_name
  • date_of_birth (YYYY-MM-DD)
  • gender (Male/Female/Other)
  • contact
  • adhar_number
Address Information (Required)
  • address_line_1, city_or_town, district, state, post_code
  • current_address_line_1, current_city_or_town, current_district, current_state, current_post_code
Bank Account (Required)
  • account_number, bank_name, ifsc_code, account_type
Loan Information (Required)
  • loan_category_name
  • loan_amount
  • tenure_value, tenure_unit (days/weeks/months/years)
  • loan_purpose
  • interest_rate
  • emi_amount
  • branch_name
EMI Schedule Information (Required for EMI Creation)
  • emi_start_date (YYYY-MM-DD)
  • emi_frequency (monthly/weekly/daily)
Application Status (Required for EMI Creation)
  • application_status (disbursed_fund_released/closed for EMI creation)
Optional Fields
  • father_name, guarantor_name, email, pan_number, voter_number
  • agent_name
  • approved_at, disbursed_at, submitted_at (Timeline dates)
  • rejection_reason, document_request_reason, ever_branch_approved
  • product_name, loan_percentage, sale_price, processing_fee, down_payment
  • disbursement_amount, disbursement_mode, disbursement_bank_name, disbursement_account_number
  • disbursement_net_amount, disbursement_tax_charges, disbursement_date, disbursement_remarks
  • emi_collected_amount, emi_principal_received, emi_interest_received, emi_payment_mode
  • emi_collected_at, emi_collected_by_agent, emi_collection_remarks, emi_status
  • branch_transaction_type, branch_transaction_amount, branch_transaction_purpose
  • branch_transaction_account_number, branch_transaction_date, branch_transaction_description
  • id_proof_path, photo_path, signature_path, income_proof_path (Document paths)
Important:

EMI schedules are only created when application_status is disbursed_fund_released or closed.

When status is closed, all EMI payments are automatically marked as completed.

Note: Branch names, loan categories, and agent names must exist in the system before import.
